# Ace Attorney Presentation

Projeto de apresentação interativa inspirado na franquia Ace Attorney, desenvolvido para apresentar o capítulo 1 do livro *Roube como um Artista* de forma criativa e dinâmica.

## Sobre o projeto

Este projeto simula um julgamento fictício onde um desenvolvedor é acusado de plagiar código. A apresentação acontece em formato visual semelhante ao jogo Ace Attorney, com diálogos, cenas, provas, objeções e participação da turma como júri.

## Objetivo acadêmico

Este projeto foi desenvolvido exclusivamente para fins educacionais e apresentação em sala de aula.
